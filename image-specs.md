## Технические параметры

| Параметр | Значение |
|----------|----------|
| Формат | WebP (предпочтительно) или JPG |
| Качество | 85% |
| Цветовое пространство | sRGB |
| Максимальный размер файла | < 500 KB |
| Фон для Quick Links | Белый + тёмно-синий + красный акцент |
| Фон для Hero слайдов | Возможен оверлей с градиентом (#001A33) |
| Стиль | Photorealistic / Product quality |

## Пути в HTML

После генерации изображений, в каждом HTML-файле заменить градиентные заглушки на:
```html
<img src="images/xxx.jpg" alt="Описание" class="w-full h-full object-cover" />
```
